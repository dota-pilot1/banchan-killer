## 현재 블로커

### 결제위젯 연동 키 필요

현재 에러:
```
결제위젯 연동 키의 클라이언트 키로 SDK를 연동해주세요.
API 개별 연동 키는 지원하지 않습니다.
```

**원인**: API 개별 연동 키(`test_ck_...`)를 사용 중. 결제위젯용 키(`test_gck_...`)가 필요.

**해결**: 토스 대시보드 → 개발 정보 → **결제위젯 연동 키** 탭 → 클라이언트 키 복사 → `.env` 교체

심사 통과와 무관하게 테스트 키는 즉시 사용 가능.
